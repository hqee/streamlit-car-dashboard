# Saudi Arabia Used Car Price Dashboard

## Deskripsi

Dashboard interaktif untuk menganalisis harga mobil bekas di Arab Saudi. Proyek ini dibuat sebagai bagian dari Mini Project Capstone.

## Fitur

- **Data Overview:** Melihat statistik data mentah.
- **EDA Interaktif:** Filter data berdasarkan Tahun dan Merek.
- **Prediksi Harga:** (Jika sudah ada model ML).

## Cara Menjalankan (Local)

1. Clone repository `git clone `
2. Install library: `pip install -r requirements.txt`
3. Jalankan aplikasi: `streamlit run app.py`
